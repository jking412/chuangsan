# 一、学生端的学习行为数据分析成果

## 1、学生用户画像设计

| 一级标签 | 二级标签                                                     |
| -------- | ------------------------------------------------------------ |
| 学习态度 | 登录次数t_stat_login；在线时长t_stat_online_time-->online_time；课堂活动参与度t_stat_activity_log；在线讨论参与度t_stat_bbs_log；任务点完成度t_stat_job_finish；作业完成度t_stat_work_answer |
| 学习效率 | 根据任务点完成度与作业完成度推算                             |
| 交流协作 | 在线讨论t_stat_bbs_log；                                     |
| 学习效果 | 作业成绩t_stat_work_answer-->score；考试成绩t_stat_exam_answer-->score;综合成绩t_stat_student_score-->score; |
|          |                                                              |

学习态度下有六个二级标签，登录次数指的是学生登录在线平台进行有效学习的次数；在线时长包括了学生在线进行课程学习、资源浏览、完成作业、测试、考试等的学习行为的总时间；任务点完成度度是指用户在确认结束课程学习时，任务点完成的百分比；课堂活动参与度、在线讨论参与度、任务点完成度指用户进行上述活动的次数与活动总次数之比。学习效率由于数据集中缺少完成某项活动的具体时间，所以只能通过构建任务点完成度、作业完成度与在线时长的关系来推算；交流协作指学生对于在线讨论的参与度；学习效果指平时的作业成绩、测试成绩、综合成绩，这些成绩指标一定程度上反映了学习者的学习效果。。。



## 2、成绩预警模型

通过学生的基础数据、学习数据、日常行为等特征维度，利用**随机森林算法**训练成绩预警模型，来预测学生的挂科情况。随机森林是采用多个决策树的投票机制构造的。决策树是用树的结构来构建分类模型，树形模型更加接近人的思维方式，可以产生可视化的分类规则，产生的模型具有可解释性。决策树中每个节点代表着一个属性，根据这个属性的划分，进入这个节点的儿子节点，直至叶子节点，每个叶子节点都表征着一定的类别，从而达到分类的目的。学生的各个特征维度信息在训练过程中，将映射成为树的节点，叶子节点则代表学生挂科或不挂科，通过训练，可以预测学生挂科的概率，从而在学习层面为学生构建更清晰的画像。

## 3、学生总评价模型

（使用CART决策树来构建模型：首先进行数据的预处理；然后进行基于CART决策树的评价模型的构造与训练；最后进行模型准确性测试与评价）。

影响因素项：任务点完成度，学习速度，考试成绩，作业成绩，在线讨论次数。

其中考试成绩所占比重最大，在线讨论次数所占比重最小。

**（* 可以以图像的形式展现出该学生各门课程的成绩以及班级平均值，可以让学生直观地了解自己的水平）**

## 4、学生偏好模型

影响因素项：学习态度（课堂活动参与度t_stat_activity_log；在线讨论参与度t_stat_bbs_log；任务点完成度t_stat_job_finish；作业完成度t_stat_work_answer），学习效率，学习效果

学习态度应占最大比重

## 2、需要教师重点关注的差生统计模型

影响因素项：学生到课率，任务点完成度，作业完成度，作业成绩，考试成绩。

到课率和考试成绩应占最大比重。

（可结合学生端的挂科预警模型，将有可能挂科的学生筛出来，重点关照）

## 3、对于课堂活动的调整与改进模型

影响因素项：各种类型的课堂活动的参与度（ t_stat_activity_log-->dtype-->AttendLog签到VoteLog  投票SurveyLog问卷WorkLog  测验ChooseSomeoneLog 选人PreampAnswerLog  抢答MarkLog  评分askLog  分组任务活动LiveLog  课堂直播CourseNoticeLog课程通知FeedbackLog  群聊SyncClassroomLog 同步课堂DiscussLog  课程讨论TimerLog 计时器活动ResourceActivitLog  资源型活动OnlineCourseLog  在线课堂OtherLog  其他活动）

对于不同类型的课堂活动参与度进行数据分析，得出学生对于不同种类的课堂活动的接受程度，给出应如何调整不同课堂活动占比的建议

## 4、课程模块掌握情况分析模型

影响因素项：任务点完成度（t_stat_job_finish）

根据学生对于各个任务点的完成度，判断学生们对于本课程各个模块的掌握情况，以图象的形式直观呈现，便于教师调整教学计划，及时安排复习。
